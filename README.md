# Nifty Intraday Trading Bot App

यह ऐप Kivy framework से बनाया गया है और इसमें डार्क थीम UI है।  
GitHub Actions से APK स्वतः build होता है।

## Features
- Simple Intraday Signal (BUY / SELL / HOLD / EXIT)
- Auto update every 10 seconds
- Dark theme interface

## Run locally
```bash
python main.py
